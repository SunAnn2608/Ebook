Решение алгоритмической задачи №121

Проверка принадлежности числа C диапазону [A, B].

Пользователь вводит три целых числа: A, B, C.

Если выполняется условие A ≤ C ≤ B, данные отправляются на сервер.

В противном случае выводится сообщение «Не принадлежит».

Сервер для отправки:

Используется учебный сервер https://httpbin.org/get,

который показывает все переданные параметры.
